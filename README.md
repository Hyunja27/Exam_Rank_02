# Exam_Rank_02
## 준비물
* A4
* 펜

## 시험시작하기
* 티켓 만들기
~~~
kinit (인트라 id)
~~~
* examshell 접속하기
~~~
examshell
~~~

## examshell 명령어
* status : 상태 보기
* grademe : 평가 받기
* finish : 끝내기

## vim설정하기
* 경로 위치 
~~~
~/.myvimrc
~~~
* 옵션 추가
~~~
set nu    // 라인 번호 
syntax on // 글자 색상
~~~
